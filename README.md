# AI 旅游攻略助手

基于 Python FastAPI + Ollama 本地大模型开发的智能旅游问答助手，实现景点推荐、行程规划、美食攻略等旅游场景对话，已打包为 Windows 独立 exe，双击即可运行。

---

## 技术栈
- 后端：Python、FastAPI、Ollama
- 前端：HTML + 原生 JS
- 交付：PyInstaller 打包为 Windows 可执行文件

## 核心功能
- 智能问答：景点推荐、行程规划、美食攻略、出行避坑指南
- 多轮对话：支持上下文记忆，对话体验连贯
- 本地部署：私有化部署大模型，数据不外流，支持离线使用
- 一键运行：适配 Windows 11，无需配置 Python 环境，双击即可启动

## 项目亮点
1. 完整的前后端开发 + 打包交付流程，从需求到落地全链路实现
2. 解决了本地服务跨平台部署问题，普通用户零门槛使用
3. 贴合当前 AI 应用开发趋势，具备私有化部署与桌面端交付能力

## 使用说明
1. 下载 `旅游助手.exe`
2. 本地安装 Ollama 并启动 `qwen2.5:7b` 模型
3. 双击运行程序，将自动打开浏览器聊天页面
